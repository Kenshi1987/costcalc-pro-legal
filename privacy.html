Privacy Policy

Last updated: March 26, 2025

1. Introduction

This Privacy Policy outlines how CostCalc Pro collects, uses, and safeguards the information provided by its users. We are fully committed to data security and transparency.

2. Information Collected

  • Images: Only those the user chooses to upload to illustrate recipes. These images are stored locally and never transmitted externally.

3. Use of Information

All data collected is strictly used to operate and improve the application's functionality. We do not share, sell, or transfer any personal data to third parties.

4. Data Security

All user data remains on their device. No remote storage or backup is performed. The user retains full control over their information.

5. Consent

By using the application, the user explicitly agrees to this Privacy Policy. If the user disagrees with any part, they should refrain from using the app.

CostCalc Pro reserves the right to modify this policy at any time. Updates will be communicated within the app or through our official release channels.
